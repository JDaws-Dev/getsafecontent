# SafeTunes Facebook Ads Guide

This document catalogs all Facebook ad materials created for SafeTunes, including carousel ads, simple ads, and Facebook page assets.

## Quick Reference

### Ad Formats
- **Carousel Ads** - 5-slide swipeable stories (3 complete series)
- **Simple Ads** - Single-image ads (2 variations)
- **Facebook Page Assets** - Profile picture and cover photo

### Best Practices
- All images are 1080x1080px (square format for mobile optimization)
- Each carousel tells a complete story: Hook → Problem/Solution → Features → CTA
- Copy is benefit-focused, not feature-focused
- Target audience: Parents (primarily moms) ages 30-45

## How to Use These Materials

### Taking Screenshots
1. Open HTML file in browser
2. **Mac:** Cmd+Shift+4, then Spacebar, click window (captures exact size)
3. **Windows:** Use Snipping Tool or Snip & Sketch
4. Save as PNG for best quality

### Uploading to Facebook Ads Manager
1. **Carousel Ads:**
   - Create new ad campaign
   - Choose "Carousel" format
   - Upload all 5 slides in order
   - Each slide is clickable → lands on getsafetunes.com

2. **Single Image Ads:**
   - Create new ad campaign
   - Choose "Single Image" format
   - Upload 1080x1080px image
   - Add headline and CTA button

### Testing Strategy
- Run all 3 carousel series simultaneously (A/B/C test)
- Track which series has highest click-through rate (CTR)
- Track which series has lowest cost per acquisition (CPA)
- Double down on winner after 1,000 impressions each

## Performance Metrics to Track

### Ad Level
- **CTR (Click-Through Rate):** Target 2-3%
- **CPC (Cost Per Click):** Target under $1.50
- **CPM (Cost Per 1,000 Impressions):** Target under $15

### Landing Page
- **Conversion Rate:** Target 5-10%
- **Bounce Rate:** Target under 50%

### Trial to Paid
- **Trial-to-Paid Rate:** Target 30-40%
- **LTV (Lifetime Value):** Target 6+ months retention
